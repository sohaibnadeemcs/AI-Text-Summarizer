# 🧠 AI Text Summarizer

A responsive web application that uses AI to instantly summarize any text into clear bullet points.

## ✨ Features
- Paste any text and get an instant AI-powered summary
- Clean overview, key bullet points, and conclusion
- Character counter
- One-click copy of summary
- Mobile responsive design
- Error handling

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Groq AI API (Llama 3.3 70B)
- Prompt Engineering

## 🚀 How to Use
1. Clone or download the repository
2. Get a free API key from https://console.groq.com
3. Add your key in `script.js` line 1
4. Open `index.html` in your browser
